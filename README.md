
# 本项目是基于vue2的实战项目

## 项目所用技术

- vue-cli
- vue
- axios
- vue-router
- better-scroll
- es6
- sass
- webpack


## 安装与运行

git clone https://github.com/shineHebin/hungry-app.git

cd hungry-app

npm install //安装依赖

npm run dev //服务端运行 访问 http://localhost:8088

npm run build  //项目打包
